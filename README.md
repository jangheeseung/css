* 사이즈 조정
	* px vs em vs rem(rem이 가장 최근에 나온 것)->절대적이냐 상대적이냐의 차이

		* px은 고정된 폰트(절대적)

		* rem과em은 가변된 폰트(상대적)

	이중 rem이 가장 좋다
	왜냐하면 폰트크기를 조정할수 있는 사용자의 권리를 보장해 준다.
